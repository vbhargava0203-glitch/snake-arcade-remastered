A modern reimagining of the classic Snake Game built using pure HTML5, CSS3, and JavaScript.

## 🎮 Live Features

- Smooth Snake Movement
- Dynamic Level Progression
- Increasing Difficulty
- Combo System
- Particle Explosion Effects
- Modern Futuristic UI
- Responsive Controls
- Keyboard Support
- Touch Swipe Controls
- Mobile Friendly
- Best Score Tracking
- Arcade Style Animations

## 📸 Preview

Modern cyber-themed Snake Game with glowing effects, animated food, particle systems, and level-based gameplay.

- Vanilla JavaScript
- Responsive Design
